Advanced Search Syntax

Easily switch between search modes without opening the dropdown menu. Simply start typing a material id, a formula, or a chemical system and submit your search. We also support the following syntax rules for specific search modes:

Formula Search Syntax

In addition to standard formula input, formulas containing wild card elements are supported.

This allows you to specify a composition with explicit elements as well as any other element that might exist in the composition.

For example, *2O will yield a list of the following formula results:

B2O, Xe2O, Li2O ...

Element Search Syntax

Li-Fe-O or *-Fe-O will be interpreted as chemical systems, with * denoting a wild card element

Mpquery Search Syntax

Using a full dict syntax, even more powerful queries can be constructed. For example, 

{"elements":{"$in":["Li", "Na", "K"], "$all": ["O"]}, "nelements":2}
selects all oxides of Li, Na and K, and 
{"band_gap": {"$gt": 1}}
selects all materials with band gaps greater than 1 eV.

Consult our API's documentation repository for descriptions of each of the properties available as search criteria.
